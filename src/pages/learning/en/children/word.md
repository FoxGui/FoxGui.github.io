## 构造单词有规律

1. 相关相仿
- 模仿旧词造新词

```
example: 
  peck -- pick
  啄食 -- 采摘，挑选

  veto -- vote
  投反对票 -- 投票

  back -- pack
  回，后背 -- 背包；打包；使聚成团

  sting -- stink
  刺 -- 刺鼻（very smelly）

  break -- brake
  中断 -- 车闸

  wine -- vine
  葡萄酒 -- 葡萄藤；藤本植物
```

2. 复合

```
example:
  blackboard
  reading-room
  bathroom
  somebody

  e-mail(electronic mail)

  blackball: 投反对票
```

3. 拼缀

```
example:
  email(electronic mail)
  slight(small + light): 轻微的
  kidult(kid + adult): 老少咸宜的
  hedge(high + edge): 树篱
  ballot(ball + lot): 投票；选票
```
