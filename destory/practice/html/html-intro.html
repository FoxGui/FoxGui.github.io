什么是HTML

超文本标记语言（英语：HyperText Markup Language，简称：HTML） 是一种用于创建网页的标准标记语言。我们可以使用 HTML来建立自己的 WEB 站点， HTML 运行在浏览器上，由浏览器来解析。


HTML功能

New:HTML5新增加的标签

标签 HTML5 描述
基础
<!DOCTYPE> 支持 定义文档类型
<html> 支持 定义一个HTML文档
<title> 支持 为文档定义一个标题
<body> 支持 定义文档主体
<h1> to <h6> 支持 定义 HTML 标题
<p> 支持 定义一个段落
<br> 支持 定义简单的折行。
<hr> 支持 定义水平线。
<!--...--> 支持 定义一个注释
格式
<acronym> 不支持 定义只取首字母的缩写
<abbr> 支持 定义一个缩写
<address> 支持 定义文档作者或拥有者的联系信息
<b> 支持 定义粗体文本
<bdi>New 支持 允许设置一段文本，使其脱离父元素的文本方向设置
